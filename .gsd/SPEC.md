# SPEC.md — Project Specification

> **Status**: `FINALIZED`

## Vision
ImpactBridge is a trust-first NGO donation platform designed to bridge the massive trust deficit in Indian philanthropy by connecting verified NGOs with donors and utilizing AI agents to provide complete milestone-by-milestone transparency into how donated funds are used.

## Goals
1. **Verifiable Transparency**: Build a robust, AI-powered milestone tracking system where NGOs submit proof of utilization (receipts, photos, beneficiary data), Gemini API validates this proof, and verified results trigger automated, personalized impact reports.
2. **NGO Accountability & Health**: Automatically track NGO credibility using an NGO Health Score recalculation engine based on utilization rates, milestone completion rates, proof submission speed, and donor retention.
3. **Immersive Donor Experience**: Create a personalized "Impact Portfolio" (Impact Ledger) for donors, showing real-time milestone progress, visual storytelling of their contributions, followed NGOs, and downloadable auto-generated 80G tax receipts.
4. **Corporate/CSR Engagement**: Provide a CSR Dashboard for corporate donors, displaying aggregate reports, compliance-ready exports, and utilization certificates.
5. **Secure Administrative Governance**: Establish a robust Admin Portal for onboarding, manual proof verification overrides, and managing platform compliance.

## Non-Goals (Out of Scope)
- **Prepaid Wallet Balance**: Holding a virtual prepaid balance for donors. All donations go directly to the NGO/project in real-time via Razorpay.
- **Automated Verification of NGO Legal Status**: Automatically calling government registries for PAN/80G status. The admin verification is manual document review.
- **In-App Payment Gateway for International Donations**: Only Razorpay (domestic Indian cards, UPI, netbanking) is in scope for v1.

## Users
1. **NGOs**: Onboard by uploading PAN, registration, and 80G documents; create projects, define milestone sequences, upload milestone completion proof (text, receipts, photos), and view health scores.
2. **Donors**: Browse/discover verified NGOs and projects, make direct donations via Razorpay, follow NGOs, track project milestones, view personal Impact Portfolios, and download 80G receipts.
3. **Admins**: Review NGO registration documents, approve/reject NGOs, review flagged/low-score AI milestone proofs, perform manual overrides, and view platform-wide analytics.

## Constraints
- **Tech Stack**: Next.js 14 App Router (monorepo), Neon serverless PostgreSQL, Prisma ORM, Google Gemini API, Local/S3/Cloudflare R2 file storage, Razorpay, NextAuth.js (JWT), Resend email, Firebase Cloud Messaging (FCM).
- **Environment Constraints**: Deployable on Vercel with serverless functions (meaning PDF generation with `@react-pdf/renderer` must stay within the 50MB function size limit).
- **Security & Validation**: Role-based access control (DONOR, NGO, ADMIN) on all API routes, Razorpay webhook signature verification, and secure file upload limits.

## Success Criteria
- [ ] 100% of successful Razorpay donations result in automated, downloadable 80G tax receipt PDF generation.
- [ ] Gemini API successfully evaluates uploaded photos/documents against milestone objectives and outputs a validation score/feedback.
- [ ] Milestones with validation scores below 70 are routed to the Admin panel, blocking automatic completion.
- [ ] Personalized donor impact narratives are successfully generated by Gemini and dispatched via Resend/FCM for every donation matching a completed milestone.
- [ ] The NGO Health Score automatically updates whenever a milestone's status is changed to completed.
